Cómo hacer una llamada a la API en una aplicación de flutter


Lo primero que debemos hacer es crear nuestro nuevo proyecto de flutter, una vez que se generen los archivos referentes a estas, debemos incluir una dependencia, para ello, nos iremos a pubspec.yaml e incluir lo siguiente: 

      
      http: ^0.13.5

Seguidamente nos iremos a nuestro main principal, en dicho archivo podremos exportar la dependencia que acabamos de incluir a nuestro proyecto: 

      
      import 'package:http/http.dart' as http;

Ahora lo que nos hace falta es incluir el código que nos permitirá poder consumir esta Api-rest, para ello simplemente vamos a incluir los siguientes archivos que se encuentran en git-hub: 

      
      https://github.com/CanulTamayAlex03/consumo_api-rest
