Cómo hacer una llamada a la API

Busqueda global

      
      https://mi-api-rest.fly.dev/api/buscar?id={id}&nombre={nombre}&img={img}&precio={precio}&fechaCracion={fechaCracion}
Ejemplo


      https://mi-api-rest.fly.dev/api/buscar?id=1&nombre=Agendas&img=https://api.com&precio=4355&fechaCracion=15-02-2023

      Sujerencias:
      Si deseas busar un producto es necesario ver la tabla de los productos y
      insertar los mismos datos a la hora de buscar un producto.
Busqueda por ID


      https://mi-api-rest.fly.dev/api/id-buscar?id={id}
Busqueda por nombre


      https://mi-api-rest.fly.dev/api/nombre-buscar?nombre={nombre}

Consumir/GET

      
      https://mi-api-rest.fly.dev/api
Insertar/POST

      
      https://mi-api-rest.fly.dev/api/nuevo

      Body:

      {
        "id":1,
        "nombre":"Ejmplo",
        "img":"url image",
        "precio":1000,
        "fechaCracion":"Dia-Mes-Año"
      }
Actualizar/PUT

      
      https://mi-api-rest.fly.dev/api/actualizar/{id}

      Body:

      {
        "nombre":"Ejemplo",
        "img":"url image",
        "precio":1000,
        "fechaCracion":"Dia-Mes-Año"
      }
Eliminar/DELETE

      
      https://mi-api-rest.fly.dev/api/eliminar/{id}

Cómo hacer una llamada a la API en una aplicación de flutter


Lo primero que debemos hacer es crear nuestro nuevo proyecto de flutter, una vez que se generen los archivos referentes a estas, debemos incluir una dependencia, para ello, nos iremos a pubspec.yaml e incluir lo siguiente: 

      
      http: ^0.13.5

Seguidamente nos iremos a nuestro main principal, en dicho archivo podremos exportar la dependencia que acabamos de incluir a nuestro proyecto: 

      
      import 'package:http/http.dart' as http;

Ahora lo que nos hace falta es incluir el código que nos permitirá poder consumir esta Api-rest, para ello simplemente vamos a incluir los siguientes archivos que se encuentran en git-hub: 

      
      https://github.com/CanulTamayAlex03/consumo_api-rest
